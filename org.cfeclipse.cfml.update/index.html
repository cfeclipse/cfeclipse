CFEclipse Update Site

This is the Eclipse update site for the CFEclipse Eclipse plugin. You need to have Eclipse version 3.4 or above, and add an update site using this URL:

http://cfeclipse.org/update/

to install it.

Installation instructions are here: https://github.com/cfeclipse/cfeclipse/wiki

Alternatively, theoretically, you can download an archived update site from here: http://cfeclipse.org/update/cfeupdatesite.zip
